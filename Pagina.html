¿Qué es y para qué sirve GIT?

Git es un sistema de control de versiones distribuido (DVCS) creado por el equipo de Linux. Actualmente lo utilizan muchos servidores de control de versiones, como GitHub, BitBucket o GitLab, para almacenar proyectos de forma remota. Pero, si queremos interactuar con estos proyectos o repositorios remotos desde nuestra máquina local, necesitamos instalar Git localmente y hacer uso de los diferentes comandos proporcionados por Git.
